# softmax-regression-concise — 练习解答

1. 调整超参数（批量大小、迭代轮数、学习率）会怎样？
   - 批量大小：增大可提升吞吐与稳定性，但过大可能泛化变差；过小噪声大但正则化效果强。
   - 学习率：过小收敛慢；适中收敛快；过大振荡/发散。可配合学习率调度（阶梯、余弦、OneCycle）。
   - 迭代轮数：适度提升可提高训练精度；过多易过拟合（见下一问）。

2. 增加训练轮数后测试精度下降的原因与对策？
   - 原因：过拟合（训练误差继续下降而验证/测试误差上升）。
   - 对策：早停（Early Stopping）、权重衰减（L2）、数据增强、Dropout、合适的学习率调度、交叉验证选择超参数。
