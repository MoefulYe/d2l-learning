# linear-regression — 练习解答

1. 假设我们有一些数据 x1,…,xn∈R。我们的目标是找到常数 b，使得最小化 ∑(xi−b)^2。
   - 解析解：b* = (1/n)∑i xi（样本均值）。
   - 与正态分布的关系：在高斯噪声（方差固定）假设下，最小二乘等价于极大似然；样本均值是正态分布均值参数的 MLE。

2. 推导使用平方误差的线性回归解析解（忽略偏置 b）。
   - 矩阵形式：最小化 L(w) = 1/2 ||Xw − y||^2。
   - 梯度：∇w L = X^T(Xw − y)。
   - 设梯度为 0 得到正规方程：X^T X w = X^T y。
   - 解析解：w* = (X^T X)^−1 X^T y（若 X^T X 可逆）；更一般为 w* = X^+ y（穆尔-彭若斯广义逆）。
   - 何时优于 SGD：样本/特征规模中小、条件数良好、需精确解时；何时失效：X^T X 奇异/病态、数据超大内存不支、需在线/增量训练、加入复杂正则/约束时。

3. 若噪声 ε ~ Laplace(0,1)（题设 p(ε)=1/2·exp(−|ε|)）。
   - 负对数似然：−log P(y|X) = 常数 + ∑i |yi − w^T xi − b|（即 L1 损失，最小绝对偏差）。
   - 解析解：常数模型时 b* 为样本中位数；一般线性情形无闭式解，可转化为线性规划或用次梯度/鲁棒回归求解。
   - SGD 思路：对残差 ri=yi−w^Txi−b 使用次梯度 sgn(ri)，更新 w ← w − η·∑i sgn(ri)xi、b ← b − η·∑i sgn(ri)。
   - 潜在问题与对策：在 ri≈0 处不可导、更新方向不稳定可振荡；可用光滑替代（Huber、SmoothL1）、添加动量/退火学习率、批量或中值截断提升稳健性。
